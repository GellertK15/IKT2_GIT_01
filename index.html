Git alapvető parancsai

git init - Git repository létrehozása az aktuális könyvtárban

git clone [url] - Egy létező repository klónozása

git status - Ha már létrejött a repository, annak állapotát (változott-e valami, stagelve van-e valami) a státusz lekérdezésével tehetjük meg

git add [fájlnév] - Stagelni fájlokat az add parancssal lehet

git commit -m "[üzenet]" - A commit parancs segítségével elmenthetjük a változtatásainkat a lokális repository-ba

git push - A push parancs segítségével feltölthetjük a lokális repository-ban lévő változtatásainkat a távoli repository-ba

git pull - A pull parancs segítségével letölthetjük a távoli repository-ban lévő változtatásokat a lokális repository-ba

git rm [fájlnév] - Fájlok eltávolítása a repository-ból
